## INNER JOIN Example

Find rentals for each inventory item

```sql
SELECT
	*
FROM
	inventory
INNER JOIN rental
ON rental.inventory_id = inventory.inventory_id
ORDER BY inventory.inventory_id
```

## LEFT JOIN Example

Find inventory items that have never been rented

```sql
SELECT
	*
FROM
	inventory
LEFT JOIN rental
ON rental.inventory_id = inventory.inventory_id
WHERE rental_id IS NULL
```

<Notes>

films witout actors:

```sql
SELECT
	*
FROM
	film
LEFT JOIN film_actor
ON film.film_id = film_actor.film_id
WHERE actor_id IS NULL
```

actors without films:

```sql
SELECT
	*
FROM
	actor
LEFT JOIN film_actor
ON actor.actor_id = film_actor.actor_id
WHERE film_id IS NULL
```

# FULL OUTER JOIN Example

Find actors withot films and films without actors

```sql

	*
FROM
	actor
LEFT JOIN film_actor
ON actor.actor_id = film_actor.actor_id
```
